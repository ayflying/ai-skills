# 微信机器人技能

## 安装命令

```bash
npx skills add ayflying/ai-skills --skill wechat-bot
```

## 功能说明

基于wxpy库的个人微信机器人，提供以下功能：

1. **消息收发**：接收和发送文本、图片、语音、视频、文件等消息
2. **自动回复**：可配置自动回复规则
3. **文件传输**：自动保存收到的媒体文件
4. **数据收集**：收集和统计消息数据
5. **命令支持**：内置help、status、stats、export等命令

## 使用指南

### 1. 安装依赖

```bash
pip install wxpy python-dotenv
```

### 2. 配置环境变量

复制 `.env.example` 为 `.env` 并修改配置：

```bash
cp .env.example .env
```

### 3. 运行机器人

```bash
# 直接运行
python scripts/wechat_bot.py

# 指定配置文件
python scripts/wechat_bot.py --config /path/to/.env

# 测试模式
python scripts/wechat_bot.py --test
```

### 4. 扫描二维码登录

首次运行会显示二维码，使用微信扫描登录。

## 命令列表

| 命令 | 说明 |
|------|------|
| `/help` | 显示帮助信息 |
| `/status` | 显示机器人状态 |
| `/stats` | 显示今日消息统计 |
| `/export` | 导出收集的数据 |

## 文件结构

```
wechat-bot/
├── SKILL.md              # 技能文档
├── scripts/
│   └── wechat_bot.py     # 主程序
├── .env.example          # 环境变量模板
├── data/                 # 数据目录（自动生成）
│   ├── sessions.json     # 会话数据
│   ├── messages_*.json   # 消息记录
│   └── media/            # 媒体文件
└── logs/                 # 日志目录（自动生成）
```

## 配置说明

### 环境变量

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `AUTO_REPLY` | `false` | 是否启用自动回复 |
| `REPLY_MESSAGE` | `收到消息，稍后回复` | 自动回复内容 |
| `COLLECT_MESSAGES` | `true` | 是否收集消息 |
| `SAVE_MEDIA` | `true` | 是否保存媒体文件 |
| `GROUP_FILTER` | 无 | 群聊过滤（逗号分隔） |
| `CONTACT_FILTER` | 无 | 联系人过滤（逗号分隔） |

### 数据存储

- **消息记录**: `data/messages_YYYY-MM-DD.json`
- **媒体文件**: `data/media/{类型}/{时间}_{发送者}.{扩展名}`
- **会话数据**: `data/sessions.json`
- **日志文件**: `logs/wechat_bot_YYYYMMDD_HHMMSS.log`

## 安全与注意事项

### 风险提示

1. **微信使用条款**：个人微信自动化可能违反微信使用条款，存在封号风险
2. **登录限制**：同一账号频繁登录可能被限制
3. **功能限制**：wxpy基于网页版微信，部分功能可能受限

### 安全建议

1. **测试账号**：建议使用测试账号进行开发
2. **频率控制**：避免频繁发送消息
3. **数据备份**：定期备份重要数据
4. **隐私保护**：不收集敏感信息

### 故障排除

| 问题 | 解决方案 |
|------|----------|
| 无法登录 | 检查网络连接，确认微信账号正常 |
| 消息收不到 | 确认机器人已成功登录 |
| 二维码不显示 | 检查终端是否支持二维码显示 |
| 发送失败 | 检查联系人是否存在 |
